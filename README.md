# Project contents

This literature dissection includes:

- photo documentation comparing spring and fall conditions at three NCOS photo points
- a related Cheadle Center poster on macroinvertebrate responses to dissolved oxygen and salinity in Devereux Slough
- an annotated bibliography connecting the project to peer-reviewed literature on dissolved oxygen, stratification, wetland restoration, salt marsh habitat, and seasonal water quality patterns

# Connection to final project

This literature dissection supports the final project questions:

- How does dissolved oxygen differ across NCOS sites?
- How does dissolved oxygen differ across elevation-code measurement positions?
- Is dissolved oxygen related to water temperature?
